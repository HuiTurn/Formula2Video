### 1. IndexError: list index out of range
- **原因**：访问列表/数组时索引超出范围
- **修复方法**：

  - 对于 VGroup，使用 `split()` 或检查子对象数量

### 2. AttributeError: VGroup object has no attribute 'vertices'
- **原因**：VGroup 是对象容器，不是几何对象，没有 `vertices` 属性
- **修复方法**：
  - 如果要从 VGroup 中获取 Polygon，先提取：`polygon = vgroup[0]` 或 `polygon = vgroup.submobjects[0]`
  - 检查对象类型：`if isinstance(obj, Polygon): vertices = obj.get_vertices()`
  - 如果 VGroup 包含多个对象，遍历查找 Polygon：`for obj in vgroup: if isinstance(obj, Polygon): ...`

### 3. AttributeError: 'X' object has no attribute 'get_vertices'
- **原因**：调用了不存在的方法或属性
- **修复方法**：
  - 对于 Polygon：使用 `get_vertices()` 或 `get_vertices()` 方法
  - 对于 Line：使用 `get_start()` 和 `get_end()` 获取端点
  - 对于 VGroup：先提取子对象再调用方法
  - 检查对象类型并使用正确的方法

### 4. SyntaxWarning: invalid escape sequence
- **原因**：字符串中包含未转义的反斜杠
- **修复方法**：

  - 或转义反斜杠：`"4 \\cdot \\frac{{1}}{{2}}ab = 2ab"`
  - 在 MathTex 中，LaTeX 公式通常使用原始字符串
